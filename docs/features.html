Features

This project provides the following features at the moment:

HST-2 Repository ResourceBundle MessageSource Integration of Spring Framework MessageSource with HST-2 Repository Dynamic ResourceBundle.
HST-2 Page Not Found Exception Handler Integration HST page-not-found sitemap item handler integration through HstBridgeHandlerExceptionsResolver bean.
Spring-session integration Easier Spring Session project integration in HST based web applications.
